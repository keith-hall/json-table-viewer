Each item in the column array is either:
- a string path (e.g., "examples.x")
- an object { "path": "examples.x", "label": "X Value", "hidden": true/false }

Examples:
[
  "Name",
  { "path": "examples.x", "label": "X Value" },
  { "path": "id", "hidden": true }
]

Omitted or hidden columns are not displayed.
